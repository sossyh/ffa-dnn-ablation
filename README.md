# FFA-DNN-Ablation

Investigating whether targeted ablation of face-relevant DNN units selectively
impairs prediction of fMRI responses in face-selective brain regions (FFA),
using the Algonauts 2021 Mini-Track dataset and AlexNet.

## Usage

Run notebooks in order:

1. `01_setup_and_data_loading.ipynb` — download data, load ROI pickles
2. `02_baseline_encoding_model.ipynb` — train baseline encoding model
3. `03_identify_face_units.ipynb` — rank AlexNet units by FFA importance
4. `04_ablation_experiment.ipynb` — ablate top units, re-evaluate
5. `05_statistics_and_controls.ipynb` — permutation tests, random-unit control
6. `06_visualization.ipynb` — heatmaps, accuracy comparison plots
